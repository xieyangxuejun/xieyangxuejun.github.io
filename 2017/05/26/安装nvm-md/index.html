安装nvm

  • 进入官网 [herfhttps://github.com/creationix/nvm]
  • curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.2/install.sh | bash
  • 安装成功 加入环境变量 我的是~/.bash_profile –> export NVM_DIR=”$HOME/.nvm”
    [ -s “$NVM_DIR/nvm.sh” ] && . “$NVM_DIR/nvm.sh”
  • 就可以使用nvm use v7.10.0..我设置默认为nvm alias default 6.10.3稳定版.
