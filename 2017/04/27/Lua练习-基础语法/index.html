Lua练习-基础语法

发表于 2017-04-27

基本数据类型

  • nil,boolean,string,number,userdata ,function,thread,table.

    通过type(print) -> function 获取类型

Lua类型装换

  • Lua会自动在string和numbers之间进行转换
    print(“10”+1) -> 11.0
    print(“hellp”+1) -> 报错
  • 当数字用连接符..数字变成字符串string
    print(10..20) -> 1020

Lua变量

  • Lua 变量有三种类型：全局变量、局部变量、表中的域。
    Lua 中的变量全是全局变量，那怕是语句块或是函数里，除非用 local 显式声明为局部变量。
    局部变量的作用域为从声明位置开始到所在语句块结束。

运算符

运算符是一个特殊的符号，用于告诉解释器执行特定的数学或逻辑运算。Lua提供了以下几种运算符类型：

  • 算术运算符
  • 关系运算符
  • 逻辑运算符
  • 其他运算符

字符串字符串或串(String)是由数字、字母、下划线组成的一串字符。

Lua 语言中字符串可以使用以下三种方式来表示：

  • 单引号间的一串字符。
  • 双引号间的一串字符。
  • [[和]]间的一串字符。

模块module

  • 模块类似于一个封装库，从 Lua 5.1 开始，Lua 加入了标准的模块管理机制，可以把一些公用的代码放在一个文件里，以 API 接口的形式在其他地方调用，有利于代码的重用和降低代码耦合度。
